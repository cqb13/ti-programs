# Geometric Sequence

Calculates values related to geometric sequences. It can determine the (n)th term (AVN), the first term (AV1), the common ratio (R), or the term number (N). The user selects the desired calculation, inputs the required values, and the program computes and displays the result based on the geometric sequence formula AVN = AV1 \* R^(N-1).
