# Sequences

This TI-Basic program for the TI-84+ calculator calculates terms and values in an arithmetic sequence. The user selects the desired calculation to find the (n)th term (AVN), the first term (AV1), the common difference (D), or the term number (N). The program prompts for the necessary values and computes the result based on the arithmetic sequence formula (AVN = AV1 + (N - 1)D).
